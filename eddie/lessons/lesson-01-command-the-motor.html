⚠ Safety Brief — read it, mean it

  • Clamp the motor. A spinning outrunner walks and jumps. Nothing on the shaft but a strip of reflective tape.
  • The whole outer bell spins, not just a shaft. Wires, fingers, and sleeves stay clear.
  • Power-up order: USB + code first (signal already saying "throttle off"), motor power (XT60) last.
  • Shutdown is the reverse. Throttle to zero in software, XT60 out, then USB.
  • Anything looks or smells wrong → pull the XT60. That kills the muscle. The brain is harmless.

Five minutes of theory

The ESC is a translator. It takes a small timing signal from the Pico and switches big current into the motor's three coils in sequence. The magnets in the bell chase the moving field. That's the whole trick.

The signal is old radio-control language: a pulse repeated 50 times per second, where the width of the pulse is the command. 1000 µs = throttle off. 2000 µs = full throttle. Everything between is proportional. The Pico's PWM hardware repeats the pulse for you at exact widths — your code just tells it how wide.

One conversion to own: the Pico wants a number 0–65535 meaning "what fraction of each 20,000 µs cycle is on." So: duty = µs ÷ 20,000 × 65,535. You'll write it once as a function and never think about it again. That's what functions are for.

Chunk 1 — Make the signal (XT60 UNPLUGGED)

# Chunk 1: generate a "throttle off" signal on pin GP15.
# The motor's big power plug (XT60) must be UNPLUGGED for this chunk.

from machine import Pin, PWM   # toolbox for controlling pins
import time                    # toolbox for waiting

# Set up GP15 as a PWM output. PWM = hardware that repeats
# a pulse over and over at a width we choose.
esc = PWM(Pin(15))

# The ESC expects the pulse to repeat 50 times per second.
esc.freq(50)

def throttle(us):
    # Convert pulse width in microseconds to the 0-65535 number
    # the Pico wants. One full cycle at 50 Hz is 20,000 microseconds.
    duty = int(us * 65535 / 20000)
    esc.duty_u16(duty)

# 1000 microseconds = "throttle off" in ESC language.
throttle(1000)

print("Signal running: 1000 microsecond pulses, 50 per second.")
print("Meter on DC volts: red probe GP15, black probe GND.")
print("Expect roughly 0.16 V - the 3.3 V pulse averaged out,")
print("because it's only on for 1/20th of each cycle.")
CHECKPOINT — Meter reads ~0.16 V between GP15 and GND. Reads 0? The signal isn't there. Reads a steady 3.3? The pulse is stuck on. Figure out which before moving.

Chunk 2 — Wake the ESC (no code — ears only)

Leave Chunk 1 running. Now plug in the XT60. You should hear the ESC's startup ritual: a few musical beeps (power good), then a confirmation tone — "I see a valid throttle-off signal, I'm armed."

CHECKPOINT — Endless beeping that never settles = the ESC can't see a valid signal. Orange wire on GP15? Brown wire on Pico GND? Chunk 1 still running?

Chunk 3 — First spin, gentle and automatic

# Chunk 3: ramp the motor up gently, hold, shut off.
# Motor CLAMPED. Stand clear of the spinning bell.

from machine import Pin, PWM
import time

esc = PWM(Pin(15))
esc.freq(50)

def throttle(us):
    duty = int(us * 65535 / 20000)
    esc.duty_u16(duty)

# try/finally is a safety promise: whatever happens inside "try" --
# an error, or you hitting Stop -- the "finally" part still runs
# and shuts the throttle off. Never run a motor without it.
try:
    throttle(1000)        # start at throttle off
    time.sleep(2)         # let the ESC settle

    print("Ramping up...")
    for pulse in range(1000, 1201, 10):   # 1000, 1010 ... 1200
        throttle(pulse)
        time.sleep(0.1)

    print("Holding at 1200 for 5 seconds. Try the tach now.")
    time.sleep(5)

finally:
    throttle(1000)        # ALWAYS end at throttle off
    print("Throttle off.")
CHECKPOINT — Smooth spin during the hold, clean stop at the end. Take one tach reading during the hold for practice. Reflective tape on the bell first.

Chunk 4 — The throttle console

# Chunk 4: type a pulse width, motor obeys. Type "stop" to quit.

from machine import Pin, PWM
import time

esc = PWM(Pin(15))
esc.freq(50)

def throttle(us):
    duty = int(us * 65535 / 20000)
    esc.duty_u16(duty)

# Safety limits for this lesson. The ESC would take 2000, but
# clamped motors at full throttle are how benches get rearranged.
LOW_LIMIT = 1000
HIGH_LIMIT = 1600

try:
    throttle(1000)
    time.sleep(2)
    print("Throttle console ready.")
    print("Number from", LOW_LIMIT, "to", HIGH_LIMIT, "or 'stop'.")

    while True:                    # loop until we break out
        answer = input("pulse width > ")

        if answer == "stop":
            break                  # jump out, land in finally

        # int() turns typed text into a number. Typing junk
        # makes it fail, so we catch that politely.
        try:
            pulse = int(answer)
        except ValueError:
            print("Not a number. Try again.")
            continue               # back to the top of the loop

        if pulse < LOW_LIMIT or pulse > HIGH_LIMIT:
            print("Out of bounds.")
            continue

        throttle(pulse)
        print("Commanded:", pulse, "microseconds")

finally:
    throttle(1000)
    print("Throttle off. Console closed.")
